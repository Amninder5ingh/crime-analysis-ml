# Crime Data Analysis and Prediction

This project analyzes crime patterns based on time and location using Python and Decision Tree Classifier.

## Steps Covered
1. Data Cleaning & Feature Engineering  
2. Exploratory Data Analysis (EDA)  
3. Visualization (Crimes by Hour, Day, Month)  
4. Decision Tree Model for Crime Time Prediction  

##  Technologies Used
- Python
- Pandas, Matplotlib, Seaborn
- Scikit-learn

##  Output
Predicts the most likely time (Morning, Afternoon, Night, Midnight) when crimes happen in a given neighborhood.
